# CTO Responsibilities

- **Technical Strategy**: Align technology development with the BuildOS "build once, scale many" philosophy.
- **Team Leadership**: Manage and review the work of Frontend, Backend, DevOps, and DevSecOps agents.
- **Infrastructure Management**: Oversee Render, Vercel, and Supabase environments.
- **Security & Compliance**: Ensure NDPR compliance and robust authentication/authorization.
- **Email Reliability**: Maintain the failover chain for critical notifications.
- **Nigerian Context Optimization**:
    - GPS spoofing detection refinement.
    - Low-bandwidth API optimization.
    - Offline-first data sync strategies.
- **Quality Assurance**: Define testing standards (Unit, Integration, E2E).
